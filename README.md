# uPomodoro

Simple pomodoro timer for the Micro::bit

## Requirements

* gcc-arm-none-eabi
* [Micro-tools](https://github.com/c272/micro-tools)
* Cmake
* Python

## Build instructions

```shell
microbuild
cp MICROBIT.hex /media/user/MICROBIT/
```
